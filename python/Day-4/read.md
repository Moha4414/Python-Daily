# Day 004 — Decision Making with Python

## Topics Covered

* Boolean Values (`True`, `False`)
* Comparison Operators (`==`, `!=`, `<`, `>`, `<=`, `>=`)
* Boolean Operators (`and`, `or`, `not`)
* Conditional Statements (`if`, `elif`, `else`)
* Nested `if` Statements
* User Input

## Exercises

### 1. Device Reachability Checker

A simple program that checks whether a network device is reachable and whether SSH access is available.

**Concepts Practiced**

* Nested conditions
* Boolean logic
* Network troubleshooting scenarios

### 2. Port Service Identifier

A program that identifies common network services based on port numbers.

**Services Included**

* SSH (22)
* Telnet (23)
* DNS (53)
* HTTP (80)
* HTTPS (443)

**Concepts Practiced**

* `if` / `elif` / `else`
* Port classification
* Network service recognition

### 3. Device Status Checker

A program that evaluates the status of a network device and provides appropriate responses.

**Statuses**

* Up
* Down
* Unknown

**Concepts Practiced**

* Conditional logic
* Input validation
* Basic network monitoring concepts

## Files

* device_reachability_checker.py
* port_service_identifier.py
* device_status_checker.py

## Learning Outcome

Learned how Python makes decisions using Boolean logic and conditional statements while applying the concepts to networking-related scenarios.
